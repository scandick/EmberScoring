# llm

- Модуль имитирует работу обращение к алгоритма расёчета уровня выгорания сотрудника. 
- Используется структурированный запрос к LLM по pydantic-модели с полями, характеризующими сотрудника, его команду и метрики. 
- Ответ LLM - структурированная pydantic модель с уровнем выгорания и рекомендациями по его снижению.
